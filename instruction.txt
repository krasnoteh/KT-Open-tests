Данный набор инструментов позволяет генерировать тесты на алгоритмические задачи по заданным паттернам
а также проверять решения по сгенерированным тестам.

Основной скрипт называется "run.py"

Рядом в той-же директории находится конфигурационный файл "config.json"

Параметры в нем:

"mode" - выбор используемого инструмента. Доступны:
    //"test" - тестировка вашего кода по созданным тестам (скоро, если будет востребовано)
    //"generate" - генерация тестов по паттерну (скоро, если будет востребовано)
    "correlate" - одновременная генерация тестов по паттерну и проверка по ним вашего кода

"labwork" - номер лабораторной работы 
"task" - номер задания (буква)
"exe_name" - название вашей скомпилированной программы (с расширением.exe) (используется только в test и correlate)
"number_of_tests" - количество генерируемых тестов (используется только в generate и correlate)
"test_size" - размер генерируемых тестов (используется только в generate и correlate)
"maximal_value" - некоторое максимальное численное значение в генерируемых тестах (используется только в generate и correlate, не всегда)



-----------
при написании паттернов тестирования, используйте класс тестера "t", предоставляющий функции:
t.add_int(a)
t.add_string(s)
t.enter()

Например, код

t.add_int(1)
t.add_int(2)
t.add_int(3)
t.enter()
t.add_string("qwe")
t.add_int(5)
t.enter()

сгенерирует 
1 2 3
qwe 5

Также можно использовать все, что предоставляет python
